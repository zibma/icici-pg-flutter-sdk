## 0.0.1 (2025-07-02)

### Added
- Initial release of the PayPhi SDK Flutter plugin
- Support for initializing the PayPhi SDK with merchant credentials
- Support for processing payments through the PayPhi payment gateway
- Stream-based payment response handling
- Comprehensive documentation and example app

### Changed
- Updated package name to `com.payphi.fluttersdkplugin`
- Refactored plugin architecture for better maintainability
- Improved error handling and logging
- Updated documentation with usage examples
